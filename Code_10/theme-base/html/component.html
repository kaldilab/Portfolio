  2. paging

    워드프레스 WP-PageNavi plug-in을 사용할 경우에 추천 방식입니다.

    wp-pagenavi
    페이징 최상의 클래스
    pages
    페이지 통계
    page
    페이지 링크
    first
    첫 페이지로 이동
    prev
    이전 페이지 묶음으로 이동
    next
    다음 페이지 묶음으로 이동
    last
    마지막 페이지로 이동
    current
    현재 페이지
    extend
    축약 페이지
    smaller
    현재 페이지보다 이전 페이지 - 잘 사용하지 않음
    larger
    현재 페이지보다 이후 페이지 - 잘 사용하지 않음

    gnb - markup

    Page 5 of 32 « First « ... 3 4 5 6 7 ... 10 20 30 ... » Last »
